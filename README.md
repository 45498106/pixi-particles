CloudKid Particles
=================

A particle system library for the PixiJS library.

##Documentation##

http://cloudkidstudio.github.io/CloudKidParticles/

##Installation##

CloudKid Particles can be installed using Bower.

```bash
bower install cloudkid-particles
```
