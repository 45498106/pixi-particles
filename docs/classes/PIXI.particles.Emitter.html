    Defined in src/Emitter.js:13

    Parameters:

    • particleParent PIXI.DisplayObjectContainer

      The display object to add the particles to.

    • [particleImages] Array | PIXI.Texture | String optional

      A texture or array of textures to use for the particles. Strings will be turned into textures via Texture.fromImage().

    • [config] Object optional

      A configuration object containing settings for the emitter.

        If config.emit is explicitly passed as false, the Emitter will start disabled.
